# Alberta Energy Pipeline

A cloud-native data pipeline that fetches daily energy data from the 
Alberta Open Government Data Portal, stores it in PostgreSQL, and 
exposes it via a REST API.

## Tech Stack
- AWS Lambda (serverless compute)
- Amazon EventBridge (daily scheduling)
- Supabase (PostgreSQL database)
- FastAPI (REST API)
- Python 3.11
